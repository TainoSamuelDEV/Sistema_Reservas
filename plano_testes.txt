4. TESTES DE RESERVA DE PASSAGENS
=======================================================================

TESTE 4.1: Reserva Válida - Classe Econômica
Objetivo: Realizar reserva com dados válidos
Passos:
1. Selecionar opção "1. Fazer Reserva"
2. Escolher voo "AA001"
3. Escolher classe "1" (Econômica)
4. Inserir nome "João Silva"
5. Inserir CPF "12345678901"
Resultado Esperado:
- Mensagem "RESERVA REALIZADA COM SUCESSO!"
- Número da reserva #1
- Dados da reserva exibidos corretamente
- Assento decrementado no voo

TESTE 4.2: Reserva com Voo Inexistente
Objetivo: Testar comportamento com código de voo inválido
Passos:
1. Selecionar opção "1. Fazer Reserva"
2. Digitar código "XX999"
Resultado Esperado:
- Mensagem "Voo não encontrado!"
- Retorna ao menu principal

TESTE 4.3: Reserva com Classe Inválida
Objetivo: Testar validação de tipo de classe
Passos:
1. Selecionar opção "1. Fazer Reserva"
2. Escolher voo válido "AA001"
3. Digitar classe "5"
Resultado Esperado:
- Mensagem "Tipo de classe inválido!"
- Retorna ao menu principal

TESTE 4.4: Reserva com Nome Inválido
Objetivo: Testar validação de nome do passageiro
Passos:
1. Fazer reserva até o passo de inserir nome
2. Digitar nome "A" (muito curto)
Resultado Esperado:
- Mensagem "Nome deve ter pelo menos 2 caracteres!"
- Retorna ao menu principal

TESTE 4.5: Reserva com CPF Inválido
Objetivo: Testar validação de CPF
Passos:
1. Fazer reserva até o passo de inserir CPF
2. Digitar CPF "123" (muito curto)
Resultado Esperado:
- Mensagem "CPF deve conter exatamente 11 dígitos!"
- Retorna ao menu principal

TESTE 4.6: Reserva com CPF Não Numérico
Objetivo: Testar validação de CPF com letras
Passos:
1. Fazer reserva até o passo de inserir CPF
2. Digitar CPF "1234567890a"
Resultado Esperado:
- Mensagem "CPF deve conter exatamente 11 dígitos!"
- Retorna ao menu principal

TESTE 4.7: Prevenção de Overbooking
Objetivo: Verificar se o sistema impede overbooking
Passos:
1. Fazer 150 reservas na classe econômica do voo AA001
2. Tentar fazer a 151ª reserva
Resultado Esperado:
- Mensagem "Não há assentos disponíveis na classe Econômica!"
- Reserva não é criada

TESTE 4.8: Múltiplas Reservas Válidas
Objetivo: Testar criação de várias reservas
Passos:
1. Fazer 3 reservas diferentes com dados válidos
2. Verificar numeração sequencial
Resultado Esperado:
- Reservas #1, #2, #3 criadas
- Cada reserva com dados corretos
- Assentos decrementados corretamente
